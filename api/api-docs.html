Endpoints

Method Path Description Notes
GET /something Get a list of Somethings. Returns array of <Something>.
  • limit (optional) (default 10) - Max number of records.
GET /something/:id Get a single Something. Returns a <Something>.
GET /anything Get a list of Anythings. Returns array of <Anything>.
  • limit (optional) (default 10) - Max number of records.
GET /anything/:id Get a single Anything. Returns a <Anything>.
GET /tag Get a list of Tags. Returns array of <Tag>.
  • limit (optional) (default 10) - Max number of records.
GET /tag/:id Get a single Tag. Returns a <Tag>.
